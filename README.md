# TODOアプリ

Reactを使ったTODOアプリケーションです。

## 起動方法

### 初回セットアップ

1. `my_app`ディレクトリに移動
```bash
cd my_app
```

2. 依存関係のインストール
```bash
npm install
```

### 開発サーバーの起動

```bash
cd my_app
npm run dev
```

開発サーバーが起動したら、ブラウザで表示されるURLにアクセスしてください。

## その他のコマンド

- `npm run build` - プロダクションビルド
- `npm run lint` - ESLintによるコードチェック
- `npm run preview` - ビルド後のプレビュー

## 技術スタック

- React 19.1.0
- TypeScript
- Vite
- TailwindCSS
- TipTap (リッチテキストエディタ)
